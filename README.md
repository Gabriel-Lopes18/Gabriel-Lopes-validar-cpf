# Validador de CPF

Projeto desenvolvido para a aula 13 de Lógica de Programação.

## Objetivo
O programa solicita um CPF ao usuário, aplica o algoritmo de validação e verifica se os dois dígitos verificadores coincidem com os informados pelo usuário, retornando se o CPF é válido ou inválido.

## Tecnologias utilizadas
- Java

## Funcionalidades
- Leitura do CPF
- Verificação da quantidade de dígitos
- Cálculo dos dígitos verificadores
- Validação do CPF
